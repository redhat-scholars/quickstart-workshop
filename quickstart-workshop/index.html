Welcome to the Console Customization Workshop

Console Customization

The purpose of this workshop is to explore several of the new OpenShift Console Customization and extensibility features.

Build clusters that are easier to use and more welcoming for new users by: 1. Enabling the Web Terminal operator 2. Learning how to offer guidance via QuickStarts 3. Customizing your Login screen 4. Configuring alerts and banners 5. Populating the default Helm repositories 6. Providing reusable YAML Samples and Code Snippets
